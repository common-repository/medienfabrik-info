A Plugin for instant support by Pfeiffer Medienfabrik via a chat icon at your dashboard page and safety warnings by Medienfabrik\'s RSS feed in a dashboard widget.

== Description ==
This plugin created by Pfeiffer Medienfabrik GmbH & Co KG contains two main functions:
-  A **Dashboard Widget** which shows the RSS-Feed from [Pfeiffer Medienfabrik](https://pfeiffer-medienfabrik.de). This feed contains articles written by staff members of Medienfabrik with regards to contents of safety hints or page approvements.
- A **Chat Icon** in the left bottom corner of your dashboard page. You can use this chat function to submit support requests or asking questions directly to staff members of  Medienfabrik. 

Via an options page you can limit excerpt length and articke count in the dashboard widget.

== Installation ==
1. Download "medienfabrik-info.zip" to your local device.
2. Unzip the file.
3. Upload the folder "medienfabrik-info" to "/wp-content/plugins/"
4. Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==
= Who will answer my support request? =
A webdevelopement staff member of Medienfabrik.

= I can't see the dashboard widget! =
On your dashboard page go to customize view and activate "Medienfabrik info".
